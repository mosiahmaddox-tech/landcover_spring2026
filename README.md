This project was developed in an internship with the United States Forest Service. As an undergraduate student studying data science and GIS at the University of Maryland, I completed the internship for academic credit as part of a course in the College of Behavioral and Social Sciences. I made this summary of land cover change to explore insights from new land cover data on Baltimore, Maryland, to visualize the land cover data, and to develop my skills in analyzing spatial data on land cover and tree canopy change. I worked with Dr. Dexter Locke in acquiring data, conducting the analysis, and writing the summary. The land cover data on Baltimore that I used was found in the Chesapeake Conservancy's LULC Viewer (link: https://experience.arcgis.com/experience/a06bf28c58994a8e9aef44f02e8051f3).

Because of the file size constraints on GitHub, the notebook for the project (tree_canopy_change.ipynb) can not be viewed on GitHub. Instead, to render the notebook displaying the project with its interactive visualizations, the URL for the notebook's page on GitHub can be entered into the search bar of NBViewer (link: https://nbviewer.org/github/mosiahmaddox-tech/landcover_spring2026/blob/main/tree_canopy_change.ipynb).
